SCK: Serial Clock - clock line, controls timing of data transmission
WS: Word Select - indicates which channel is being transmitted (left or right in stereo audio) 
SD: Serial Data - data line


- synchronous
- various possibilities of who controls the single lines in the bus 
- alternatively central "system" controller providing external clock


- transmisssion of data is synchronized to the clock signal

- can handle different word lenghts on receiver, transmitter (and system) sides => data encoding is two complement with most significant bit first (allows for truncation)

- data synchronized with either trailing HIGH-to-LOW or leading LOW-to-HIGH edge of clock signal
  
- word select line changes ones period before the MSB is transmitted
